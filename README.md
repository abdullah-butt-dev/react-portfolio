## 📊 Data Analyst Portfolio | Interactive Dashboard Showcase

A modern, responsive portfolio website built with React to showcase data analytics projects, including interactive dashboards, SQL analysis, and machine learning applications.

### ✨ Live Preview
👉 **[View Live Portfolio](https://abdullahbutt.dev)**

---

## 🎯 Features

- **📱 Fully Responsive** - Seamless experience on all devices
- **🌓 Dark/Light Mode** - Toggle between themes
- **📊 Project Showcases** - Interactive project cards with detailed insights
- **🔗 Live Demo Links** - Clickable project previews to live dashboards
- **🏷️ Tech Stack Tags** - Visual representation of technologies used
- **📈 Impact Metrics** - Key statistics for each project
- **🎨 Modern UI** - Clean, professional design with smooth animations
- **⚡ Fast Loading** - Optimized with lazy loading for images

## 🛠️ Tech Stack

### Frontend
- **React** - UI Library
- **CSS3** - Custom styling with CSS variables
- **Lucide React** - Icon library

### Hosting
- **Vercel** - Deployment and hosting

### Analytics Projects Showcased
- **Python** - Data processing and analysis
- **Pandas** - Data manipulation
- **Plotly/Seaborn** - Data visualization
- **Streamlit** - Interactive dashboards
- **SQL/PostgreSQL** - Database queries and analysis
- **Scikit-learn** - Machine learning models
